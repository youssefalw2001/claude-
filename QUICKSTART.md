## 🔑 Getting Your API Key

⚠️ **IMPORTANT:** Your old key was posted publicly. Get a new one!

**Steps:**
1. Go to: https://chat.thesys.dev/console/keys
2. Sign in (or create account - it's free)
3. Click "Create New API Key"
4. Give it a name (like "Fox AI")
5. Copy the key (starts with `sk-th-`)
6. **Delete the old key** if you see it

**Free Tier Limits:**
- Generous for personal use
- Should handle all your dev work
- Can upgrade later if needed
